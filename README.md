# Noty
Um aplicativo mobile simples na criação de notas

# 🧠 Objetivos do projeto
Entender um pouco do uso do *React Native* e seus frameworks, como o *Expo*.

# ⚙️ Ferramentas e Frameworks utilizados 
- React Native + Expo
- Style Components
- Async storage
